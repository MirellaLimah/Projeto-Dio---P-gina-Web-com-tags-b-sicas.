Estrutura Básica do HTML

(Voltar)

O HTML (HyperText Markup Language) é a linguagem padrão para criar páginas web. Sua estrutura básica inclui a declaração do tipo de documento, a tag , e dentro dela, as seções e . O contém metadados e informações como o título da página, links para arquivos CSS, e outras configurações importantes. Já o contém o conteúdo visível da página, como textos, imagens, links e outros elementos.

Entendendo as Tags

(Voltar)

Atributos das Tags

Tag <a> (Anchor - Âncora)

A tag <a> é usada para criar links. Os atributos comuns incluem:

href: Define o URL para o qual o link aponta.
target: Define onde abrir o link. Os valores comuns são _self (abre na mesma aba) e _blank (abre em uma nova aba).

Exemplo:
<a href="https://www.example.com" target="_blank">Visite Example</a>

Tag <img> (Image - Imagem)

A tag <img> é usada para inserir imagens. Os atributos comuns incluem:

src: Define o caminho da imagem.
alt: Fornece um texto alternativo para a imagem, usado para acessibilidade e quando a imagem não pode ser exibida.
width: e height: Definem a largura e a altura da imagem, respectivamente.

Exemplo:
<img src="imagem.jpg" alt="Descrição da imagem" width="300" height="200">

Tag <input> (Input - Entrada)

A tag <input> é usada em formulários para criar campos de entrada. Os atributos comuns incluem:

type: Define o tipo de campo (por exemplo, text, password, submit, checkbox, etc.).
name: Define o nome do campo, que é enviado com o formulário.
value: Define o valor inicial do campo.
placeholder: Fornece um texto de dica exibido dentro do campo quando está vazio.

Exemplo:
<input type="text" name="nome" placeholder="Digite seu nome">
<input type="password" name="senha" placeholder="Digite sua senha">
<input type="submit" value="Enviar">

Tag <button> (Button - Botão)

A tag <button> é usada para criar botões clicáveis. Os atributos comuns incluem:

type: Define o tipo de botão (por exemplo, button, submit, reset).
onclick: Define uma função JavaScript a ser executada quando o botão é clicado.

Exemplo:
<button type="button" onclick="alert('Botão clicado!')">Clique aqui</button>
<button type="submit">Enviar Formulário</button>

Tag <form> (Formulário)

A tag <form> é usada para criar formulários de entrada de dados. Os atributos comuns incluem:

action: Define a URL para onde os dados do formulário serão enviados.
method: Define o método HTTP a ser usado ao enviar o formulário (GET ou POST).

Exemplo:
<form action="/submit_form" method="post">
  <input type="text" name="nome">
  <input type="submit" value="Enviar">
</form>

Tag <div> (Division - Divisão)

A tag <div> é usada como um contêiner genérico para agrupar outros elementos. Atributos comuns incluem:

id: Define um identificador único para o elemento.
class: Define uma ou mais classes de CSS para o elemento.

Exemplo:
<div id="container" class="content-box">
  <p>Este é um parágrafo dentro de uma div.</p>
</div>

Tag <span> (Span - Intervalo)

A tag <span> é usada para estilizar partes de um texto ou agrupar elementos inline. Os atributos comuns incluem:

id: Define um identificador único para o elemento.
class: Define uma ou mais classes de CSS para o elemento.

Exemplo:
<p>Este é um texto com uma <span class="highlight">palavra destacada</span>.</p>

Estruturando Textos no HTML

(Voltar)

O HTML oferece várias tags que permitem estruturar e organizar textos de maneira eficaz dentro de uma página web. Abaixo estão algumas das principais tags e suas funções:

Tag <p> (Parágrafo)

A tag <p> é usada para definir parágrafos. Um parágrafo é um bloco de texto que geralmente contém uma ideia completa. No HTML, os parágrafos são automaticamente seguidos por uma linha em branco, o que os separa dos outros elementos.

Exemplo:
<p>Este é um parágrafo de exemplo.</p>

Tag <br> (Linha Quebrada)

A tag <br> é usada para inserir uma quebra de linha. Ao contrário da tag <p>, a tag <br> não cria um novo bloco de texto, mas simplesmente move o texto após a tag para a linha seguinte.

Exemplo:
Primeira linha de texto<br>Segunda linha de texto

Tag <h1> a <h6> (Cabeçalhos)

As tags <h1> a <h6> são usadas para definir cabeçalhos, que são títulos ou subtítulos de diferentes níveis. <h1> é o nível mais alto e geralmente é usado para o título principal da página, enquanto <h6> é o nível mais baixo.

Exemplo:
<h1>Título Principal</h1>
<h2>Subtítulo</h2>
<h3>Sub-subtítulo</h3>

Tag <strong> e <b> (Negrito)

As tags <strong> e <b> são usadas para destacar texto em negrito. A tag <strong> indica que o texto tem importância especial, enquanto a tag <b> simplesmente estiliza o texto em negrito sem nenhuma implicação adicional.

Exemplo:
<strong>Texto importante</strong>
<b>Texto em negrito</b>

Tag <em> e <i> (Itálico)

As tags <em> e <i> são usadas para estilizar texto em itálico. A tag <em> indica que o texto deve ser enfatizado, enquanto a tag <i> apenas estiliza o texto em itálico.

Exemplo:
<em>Texto enfatizado</em>
<i>Texto em itálico</i>

Tag <blockquote> (Citação)

A tag <blockquote> é usada para definir uma citação em bloco. Esse tipo de citação é geralmente exibido com um recuo em relação ao resto do texto, destacando-a.

Exemplo:
<blockquote>Esta é uma citação em bloco.</blockquote>

Tag <ul> e <ol> (Listas)

As tags <ul> (lista não ordenada) e <ol> (lista ordenada) são usadas para criar listas de itens. A tag <ul> cria uma lista com marcadores, enquanto a tag <ol> cria uma lista numerada.

Exemplo de lista não ordenada:
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

Exemplo de lista ordenada:
<ol>
  <li>Primeiro item</li>
  <li>Segundo item</li>
</ol>

Tag <span> (Span)

A tag <span> é usada para agrupar partes de um texto para fins de estilização ou manipulação via JavaScript. Diferente das tags de bloco, como <div>, a tag <span> não cria uma nova linha.

Exemplo:
<span class="highlight">Texto destacado</span>

Listas Ordenadas e Não Ordenadas

(Voltar)

O HTML permite a criação de listas para organizar e exibir itens de maneira ordenada ou não ordenada. Existem três tipos principais de listas em HTML: listas não ordenadas, listas ordenadas e listas de descrição. Cada tipo de lista serve para diferentes propósitos e pode ser estilizado de várias maneiras.

Listas Não Ordenadas (<ul>)

As listas não ordenadas são usadas para agrupar itens sem uma ordem específica. Cada item dentro de uma lista não ordenada é precedido por um marcador (ponto). A tag principal para criar uma lista não ordenada é <ul>, e cada item da lista é definido pela tag <li>.

Exemplo de uma lista não ordenada:

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
Listas Ordenadas (<ol>)

As listas ordenadas são usadas para agrupar itens em uma ordem específica. Cada item dentro de uma lista ordenada é precedido por um número ou letra. A tag principal para criar uma lista ordenada é <ol>, e cada item da lista é definido pela tag <li>.

Exemplo de uma lista ordenada:

<ol>
  <li>Primeiro item</li>
  <li>Segundo item</li>
  <li>Terceiro item</li>
</ol>
Listas de Descrição (<dl>)

As listas de descrição são usadas para listar termos e suas descrições. A tag principal para criar uma lista de descrição é <dl>. Dentro dela, cada termo é definido pela tag <dt> (definição de termo), e cada descrição é definida pela tag <dd> (descrição de definição).

Exemplo de uma lista de descrição:

<dl>
  <dt>HTML</dt>
  <dd>Linguagem de marcação usada para criar páginas da web.</dd>
  <dt>CSS</dt>
  <dd>Linguagem de estilo usada para definir a aparência das páginas da web.</dd>
</dl>

As listas são ferramentas essenciais para organizar e estruturar conteúdo de forma clara e acessível. Elas podem ser estilizadas com CSS para melhorar a aparência e a usabilidade das páginas da web.

Criando Links

(Voltar)

Links são elementos essenciais em páginas web, permitindo a navegação entre diferentes páginas e recursos na internet. Em HTML, os links são criados usando a tag <a> (âncora), que define uma ligação para outra página ou recurso. Aqui estão algumas formas de criar links e suas utilizações.

Link Básico

O exemplo mais simples de um link é aquele que leva o usuário a outra página web. A tag <a> utiliza o atributo href para especificar o destino do link.

Exemplo de um link básico:

<a href="https://www.example.com">Visite o Example</a>

No exemplo acima, o texto "Visite o Example" será exibido como um link que, quando clicado, levará o usuário ao site "https://www.example.com".

Link Interno

Links internos são usados para navegar dentro da mesma página. Eles podem ser úteis para criar um índice ou uma navegação que leva a diferentes seções da página.

Para criar um link interno, você precisa de um id no elemento de destino e um link que aponte para esse id.

Exemplo de um link interno:

<a href="#secao1">Ir para a Seção 1</a>
...
<h2 id="secao1">Seção 1</h2>

No exemplo acima, o link "Ir para a Seção 1" levará o usuário à seção da página marcada com o id="secao1".

Link para Download

Links também podem ser usados para permitir o download de arquivos. Para isso, é comum utilizar o atributo download na tag <a>.

Exemplo de um link para download:

<a href="arquivo.zip" download>Baixar Arquivo</a>

No exemplo acima, o link "Baixar Arquivo" permitirá que o usuário baixe o arquivo "arquivo.zip" ao invés de abrir o arquivo no navegador.

Link de E-mail

Você pode criar links que abrem o cliente de e-mail do usuário para enviar uma mensagem para um endereço específico usando o prefixo mailto: no atributo href.

Exemplo de um link de e-mail:

<a href="mailto:exemplo@email.com">Enviar e-mail</a>

No exemplo acima, clicar no link "Enviar e-mail" abrirá o cliente de e-mail do usuário com uma nova mensagem endereçada a "exemplo@email.com".

Link com Atributo target

O atributo target especifica onde abrir o documento vinculado. O valor mais comum para este atributo é _blank, que abre o link em uma nova aba ou janela do navegador.

Exemplo de link com target="_blank":

<a href="https://www.example.com" target="_blank">Abrir Example em nova aba</a>

No exemplo acima, clicar no link "Abrir Example em nova aba" abrirá o site "https://www.example.com" em uma nova aba do navegador.

Os links são ferramentas poderosas em HTML que permitem uma navegação eficiente e intuitiva na web. Ao aprender a criar e usar links de diferentes tipos, você pode melhorar significativamente a usabilidade e a funcionalidade de suas páginas web.
